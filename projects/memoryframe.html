Memory Frame

Memory Frame is a voice controlled digital photo frame.

The photos are first uploaded via our android app. This let’s the user control what is shown in the frame anywhere they go.

User’s can then tell Memory Frame to display any photos they want to see in any given moment. By saying phases such as, “Show me a photo of Johnny”, allowing for a personalized and familiar experience.

My Work

I was responsible for the photo frame, consisting of a Raspberry Pi and a LED screen. The photos are displayed by a web app that communicates with a local server. User’s voice commands are processed with Alexa Voice Service then sent to the local server. The local servers then tell the web app which photo to display.
